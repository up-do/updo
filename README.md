# <img src="updo-offset.svg" height="80px" />
> A hairstyle and a way of maintaining and upgrading Haskell projects.

Updo is a project generator for Haskell, making projects from packages,
versions[^constraints] and commits[^unpublished-packages].

[^constraints]: A dependency version equality constraint.
[^unpublished-packages]: A package not published to hackage or stackage.

# Docs

Documentation, including installation and usage can be found on the
[website](https://up-do.github.io/updo).
